🔑 Login com Email

Autenticacao com BCrypt no backend. Retorna JWT access token (15min) e refresh token (7 dias).

🅐 Google OAuth

Login social com Google. Se for primeiro acesso, redireciona para completar o registro com condominio e unidade.

🔄 Refresh Automatico

Axios interceptor detecta 401, usa refresh token para renovar. Queue de requests pendentes para evitar race conditions.

🚪 Redirect Inteligente

Apos login: Admin vai para /admin, Sindico vai para /c/{id}, Morador vai para dashboard do condominio.

3
Analise e Resposta
Sindico muda para "Em Analise", adiciona respostas publicas (visivel ao morador) ou internas (so staff).
4
Resolucao
Status final: RESOLVIDA ou FECHADA. Timeline completa com todas as interacoes.

Gestao de Encomendas

Porteiro recebe, gera codigo 2FA, morador retira com verificacao segura.

1
Porteiro Recebe
Registra: unidade, destinatario, transportadora, rastreio. Pode tirar foto da etiqueta.
2
Gera Codigos
Sistema gera pickup code (porteiro) e resident code (morador). Pode gerar QR Code para leitura rapida.
3
Notifica Morador
Morador recebe notificacao (in-app, email futuro). Ve o status na aba "Minhas Encomendas".
4
Verificacao 2FA
Morador apresenta seu codigo. Porteiro insere ambos os codigos no sistema. Match = entrega confirmada.
